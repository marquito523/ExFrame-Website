    Moderation

    Moderation Commands Fun Commands Music Commands Other Commands

    Moderation Commands:


    Kick
    Use: m!kick + mention member
    Does: Kicks a member, with or without a reason.
    Requires: Requires kick members permission, and member to kick under you, and without Administration permissions.

    Ban
    Use: m!ban + mention member
    Does: Bans a member, with or without a reason.
    Requires: Requires ban members permission, and member to ban under you, and without Administration permissions.

    Warn
    Use: m!warn + mention user + reason
    Does: Warns a user
    Requires: Manage guild or Administration permissions.

    Slow Mode
    Use: m!slow mode + time in seconds
    Does: Sets slow mode to channel.
    Requires: Requires Manage guild permissions.

    Filter
    Use: m!enable-filter / m!disable-filter
    Does: Filters every message, when a swear word is detected, message is deleted, and a warn is sent to the author.
    Requires: Requires manage guild

    Join Message
    Use: m!join-message + message
    Does: It will DM that member the message you set, as a welcome.
    Requires: Requires Manage guild permissions.

    Default Role
    Use: m!set-default-role + @role
    Does: Will add that role to any new members that join the server.
    Requires: Requires Manage guild permissions.

    Add Role / Remove Role
    Use: m!addrole @role @member/ m!removerole @role @member
    Does: Will add or removed the role to the member.
    Requires: Requires Manage Roles permissions.

    Change Prefix
    Use: m!prefix [new prefix]
    Does: Will change the prefix to whatever you like.
    Requires: Requires Manage guild permissions.

    Purge
    Use: m!purge [amount of message (between 1 and 100)]
    Does: Deletes a requested amount of messages.
    Requires: Requires Manage messages permissions.

    Join Notification
    Use: m!enable-join-notif + #channel
    Does: Sends a message when a user joined to the channel requested.
    Requires: Requires Manage guild permissions.

    Suggestion Setup
    Use: m!setupsuggestion + channel
    Does: Will set the requested channel as channel for suggestions.
    Requires: Requires Manage guild permissions.
